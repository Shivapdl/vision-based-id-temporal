# Dataset

The TARA dataset (89,944 RGB-D frames over 96 hours, 9 group-housed sows,
collected at the University of Arkansas Animal Research Facility under IACUC
protocol #25059) is **available on request**.

## Requesting access

Email **shivap@uark.edu** with a brief description of your intended use.
You will receive a download link and any data-use terms.

## Expected layout after download

Place the data so it matches the paths in `configs/default.yaml`:

```
data/
├── raw/                  # original captures, organized by day
│   ├── day1/
│   ├── day2/
│   ├── day3/
│   └── day4/
└── processed/            # generated by Segment_with_individual_calibration.ipynb
    ├── day1/
    └── ...
```

Each day folder contains per-visit point cloud sequences. The preprocessing
notebook reads from `data/raw/` and writes normalized 1,500-point dorsal
manifolds to `data/processed/`.

## Using your own data

You do **not** need the original dataset to run TARA. To use your own
group-housed livestock captures:

1. Place your point clouds under `data/raw/` following the layout above.
2. Set `data.num_classes` in `configs/default.yaml` to your number of
   individuals.
3. Run the preprocessing notebook, then training, then evaluation.

The pipeline expects depth-derived point clouds; see the preprocessing
notebook for the exact input format (origin-centered, unit-sphere normalized
after voxel downsampling and ROI cropping).

# Vision-Based Identification with Temporal Information





\## Overview

This project implements a deep learning pipeline for individual animal identification using vision-based features and temporal aggregation.



The framework integrates:

\- 3D point cloud processing

\- PointNet-based architecture

\- Visit-level temporal aggregation

\- Cross-day validation



\## Key Features

\- 1500-point sampling strategy

\- Individual-level classification

\- Next-day validation evaluation

\- Temporal matching across visits



\## Repository Structure

\- PointNet\_1.5K\_V5.ipynb – Training pipeline

\- Next\_Day\_Validation.ipynb – Cross-day evaluation

\- ID\_And Visit\_Match.ipynb – Temporal matching

\- Segment\_with\_individual\_calibration.ipynb – Preprocessing
